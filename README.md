# MyCompany

Проект на ASP.NET Core MVC.

Это сайт компании с админ-панелью, списком услуг, страницей услуги, авторизацией и загрузкой изображений.

## Возможности проекта

- отображает главную страницу, список услуг и контакты;
- отображает страницу одной услуги;
- позволяет войти в админ-панель;
- позволяет добавлять, редактировать и удалять категории услуг;
- позволяет добавлять, редактировать и удалять услуги;
- позволяет загружать картинки для услуг;
- использует логирование действий;
- запускается в Docker.

## Стек

- C#
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- ASP.NET Core Identity
- Serilog
- Razor Views
- HTML, CSS, SCSS, JavaScript
- TinyMCE
- Docker

## Архитектура проекта

- `Controllers` — обрабатывают запросы;
- `Views` — страницы сайта;
- `Domain` — сущности и интерфейсы для работы с данными;
- `Infrastructure` — вспомогательные классы;
- `Models` — модели для страниц и передачи данных;
- `wwwroot` — статические файлы проекта.

## Как запустить

1. Установить .NET SDK и SQL Server.
2. Настроить строку подключения в `appsettings.json`.
3. Применить миграции:
   ```bash
   dotnet ef database update
   ```
4. Запустить проект:
   ```bash
   dotnet run
   ```
